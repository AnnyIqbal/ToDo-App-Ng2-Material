# To Do App using Angular2
Bootcamp Task1

###Functionalities: 

* Sign In / Sign Out
* Create Account
* Add Task
* Delete Task 
* Edit Task

###Technologies used: 

* Angular2
* Typescript2
* ng2Material
* AngularFire2
* Auth Guard
* Redux <State Management for Tasks and Users>
